#1 株式市場

##1-1お金の時間的価値 お金は現在と将来で、価値が違う可能性がある。 将来価格をFV、現在価格をPV、利率をr、期間をnとすると、

\[FV = PV × (1 + r)^n\] \[PV = \frac{FV}{(1 + r)^n}\] のようになる。このモデルは離散時間モデルといわれる。

次に、連続的な時間でのモデルを考える。 時刻tにおけるある資産の価値をx(t)とする。利率をrとすると、微小時間\(dt\)での増分は利率と元本に比例するので \[\frac{dx}{dt} = r ⋅ x(t)\] という微分方程式が成り立つ。 これを解くと、 \[FV = PV⋅e^{rt}\] \[PV = FV⋅e^{-rt}\] と求まる。 以上のことから、離散時間モデルでも連続時間モデルでも、お金は指数関数的に成長することが言える。

##1-2 Pythonによる現在価値と将来価値の計算

離散時間モデル

投資額を X、利率を r、年数を n とすると、

\[FV = X \times (1+r)^n\] \[PV = \frac{X}{(1+r)^n}\]

連続時間モデル

時間を t とすると、

\[FV = X \cdot e^{rt}\] \[PV = X \cdot e^{-rt}\]

##1-3 株式（Stocks / Shares）

株式とは

  • 株式は会社の所有権の一部を表す。
  • 保有者は会社の利益や資産の一部を得る権利を持つ。
  • 企業は資金調達のために株式を発行する。
  • 「Stocks」と「Shares」は同義。

株式市場の仕組み

  • 企業：成長・資金調達のために株式を発行。
  • 証券取引所：企業と投資家をつなぐ場（例：NYSE、Nasdaq）。
  • 投資家：株式に投資して利益を狙う。
  • 証券会社（ブローカー）：投資家と取引所を仲介。近年はオンライン取引が主流。

株価の変動要因

  • 株価は「需要と供給」で決まる。
  • 買いが多ければ株価↑、売りが多ければ株価↓。
  • 株価推移は「ランダムウォーク」に近い。

株を持つメリット

  1. キャピタルゲイン（値上がり益）
    • 株価上昇時に売却し利益を得る。
    • 例：Tesla株（2017年 約60ドル → 2021年 約900ドル）。
  2. 配当金（Dividends）
    • 企業の利益に応じて株主に分配。
    • 四半期または半年ごとに支払われる。

株のリスクとボラティリティ

  • 株はリスクの高い資産。
  • ボラティリティ：リターンの散らばりを示す統計指標。
    • 測定：標準偏差・分散。
    • 高ボラティリティ = 高リスク。
  • 将来学ぶ内容：CAPM、β値でのリスク評価。

##1-4 コモディティ（Commodities）

コモディティとは

  • 金、銀、石油、天然ガスなどの原材料・資源。
  • 価格変動が大きく、ボラティリティが高い。
  • 株式のように簡単に「現物」を所有できないため、取引は複雑。

価格変動の要因

  • 基本は 需給（供給と需要）。
  • 買いが増えれば価格↑、売り・供給が増えれば価格↓。
  • 推移は「ランダムウォーク」に近い。

先物契約（Futures）

  • 価格変動リスクを回避するための契約。
  • 例：航空会社は燃料費変動を避けるため原油先物を購入。
  • 先物市場：ニューヨーク・マーカンタイル取引所（NYMEX）など。

投資方法

  1. 直接投資：先物市場やブローカー経由で取引。
  2. 間接投資：コモディティに依存する企業の株を購入。
    • 例：金価格 ↔︎ 金鉱株（正の相関が多い）。

ポートフォリオにおける役割

  • インフレ対策：インフレ時に商品価格は上がりやすい。
  • 分散投資効果：株・債券・コモディティを組み合わせるのが有効。
  • ペアトレード戦略：コモディティと株価の相関を利用して利益を狙う。

👉 コモディティはリスクは高いが、インフレ防衛・分散投資の観点で重要な資産クラス。

##1-5 通貨と外国為替市場（Forex Market）

為替レートとは

  • 1通貨を別の通貨に交換する比率。
  • 値 > 1 → 行側の通貨が強い、値 < 1 → 列側の通貨が強い。
  • 例：1ユーロ = 1.35ドル → ユーロ高。

為替市場（Forex）

  • Forex = Foreign Exchange（外国為替）。
  • 国際通貨（USD, EUR, JPY, CHF など）を取引する市場。
  • 投資家は FXブローカーを介してアクセス。

為替レートに影響する要因

  1. 金利（Interest Rates）
    • 中央銀行が操作可能。
    • 金利↑ → 通貨需要↑ → 為替レート上昇。
  2. マネーサプライ（Money Supply）
    • 通貨供給↑（刷りすぎ）→ インフレ → 通貨安。
  3. 金融安定性（Financial Stability）
    • 政治・経済が安定している国ほど通貨は強い。

アービトラージ（Arbitrage）

  • 複数通貨間の交換で リスクなしに利益を得る取引。
  • モデル化：通貨 = ノード、為替レート = エッジ。
  • 変換：エッジ重み = -log(rate)。
  • Bellman-Ford アルゴリズムで負のサイクルを検出 → アービトラージ機会。
  • 実際は高頻度取引（HFT）で活用される。

👉 為替市場では「需給・金利・金融安定性」が通貨価値を決める。
👉 アービトラージは理論的に可能だが、現実ではごく稀でHFTの領域。

##1-6 ロングポジションとショートポジション

ロングポジション（Long Position）

  • 株式や債券を購入・保有する。
  • 価格上昇で利益、価格下落で損失。
  • 最大損失 = 投資額（100%）

ショートポジション（Short Position）

  • 株式を売ることで価格下落を狙う。
  • 2種類：
    1. 保有株を売却 → 安値で買い戻し。
    2. 空売り（Short Selling）：株を借りて売却 → 後に買い戻して返却 → 差額が利益。
  • 株価上昇で損失、損失に上限なし（無限大）。
  • ロングよりリスクが高い。

市場環境

  • Bull Market（強気相場）：価格上昇が持続 → ロングが有効。
  • Bear Market（弱気相場）：価格下落が持続 → ショートが有効。
  • 相場はブル期とベア期を繰り返す。

👉 ロングは「価格上昇で利益」、ショートは「価格下落で利益」。
👉 ただし、ショートのリスクはロングより圧倒的に高い。
